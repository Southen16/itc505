Challenges and how they were overcome

1. Styling challenges: Achieving a visually appealing design while remaining responsive was a challenge. Through the use of flexbox and grid layout, and careful color selection, we were able to create a visually pleasing interface.

2.Randomization logic:It was difficult to randomize the original layout while still allowing the puzzle to be solved. Iterative testing and refinement of the randomization function were essential to finding a balance between randomness and solvability.

3 Timer implementation: Including a timer that starts on page load and restarts every time the game is reset required careful consideration of the timer lifecycle. . Using a combination of 'setInterval' and 'clearInterval' solved this challenge.
